Script that will output the best move for someone playing connect4 using the
Minimax algorithm.

Many parts of this script were inspired by a program written by Clederson Cruz
that uses minimax to play Tic-Tac-Toe. His work can be found here:
https://github.com/Cledersonbc/tic-tac-toe-minimax
